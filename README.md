## 🍳 서비스 소개

**냉장고 셰프**는 사용자가 보유한 재료나 냉장고 속 식재료를 입력하면, AI가 수만 개의 레시피 데이터베이스에서 최적의 요리를 추천해주는 서비스입니다.

단순한 재료 검색을 넘어 **유통기한 관리**, **AI 레시피 생성**, **단계별 조리 가이드**까지 제공하여 매일 반복되는 "오늘 뭐 먹지?"를 해결합니다.

---

## ✨ 주요 기능

### 🔍 AI 레시피 추천
- 재료를 입력하면 AWS Bedrock의 `Titan Text Embeddings`로 벡터 변환
- 수만 개의 레시피 DB에서 유사도 기반 검색
- `Claude 3 Sonnet`이 검색된 레시피를 기반으로 맞춤 레시피 생성
- **매칭 스코어**로 재료 일치율 시각화

### 🧊 냉장고 관리
- 재료별 **수량**, **단위**, **카테고리**, **유통기한** 입력 및 관리
- 유통기한 상태에 따른 색상 구분 (여유 🟢 / 임박 🟡 / 만료 🔴)
- 임박·만료 재료 경고 배너 및 네비게이션 뱃지 알림
- "냉장고 재료로 레시피 찾기" — 저장된 재료를 그대로 레시피 검색에 활용

### 📖 레시피 상세
- 단계별 조리법 (클릭으로 완료 체크, 진행률 바)
- 재료 목록 (보유 재료 하이라이트)
- 영양 정보 (칼로리 / 단백질 / 탄수화물 / 지방)

### 🔎 레시피 탐색
- 카테고리 / 난이도 / 칼로리 슬라이더 필터
- 조리시간·칼로리·난이도순 정렬
- 그리드 / 리스트 뷰 전환

### 🔖 즐겨찾기
- 레시피 북마크 저장, 카테고리별 분류 표시

### 🔐 회원 인증
- 이메일/비밀번호 로그인 및 회원가입
- 실시간 유효성 검사 + 비밀번호 강도 측정
- 비로그인 시 서비스 접근 제한

## 🛠 기술 스택

### Frontend
| 분류 | 기술 |
|------|------|

| Language | TypeScript |

|------|------|

| 분류 | 기술 |
|------|------|
| AI Platform | AWS Bedrock |

## 🤖 RAG 아키텍처

```
사용자 입력 ("감자, 스팸")
        │
        ▼
┌───────────────────┐
│  Titan Embeddings │  ← 텍스트 → 벡터 변환
└────────┬──────────┘
         │ 벡터
         ▼
┌───────────────────┐
│    Vector DB      │  ← 유사 레시피 Top-K 검색
└────────┬──────────┘
         │ Context (레시피 3~5개)
         ▼
┌───────────────────────────────────┐
         Prompt Augmentation   

     System: "너는 전문 셰프야..."      
     Context: (검색된 레시피)           
     Query: "감자랑 스팸으로 만들 수     
             있는 요리 알려줘"         
└────────┬──────────────────────────┘
         │
         ▼
┌───────────────────┐
│   Claude 3 AI     │  ← 최종 레시피 생성
└───────────────────┘
```

## 🚀 시작하기

### 요구사항
- Node.js 18+
- npm

### 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/0wkim/2026_ACC_BedRock_Project.git
cd 2026_ACC_BedRock_Project

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

개발 서버가 실행되면 [http://localhost:5173](http://localhost:5173) 에서 확인할 수 있습니다.

### 빌드

```bash
npm build
```
